# Multi-Class Emotion Classification
This project details the implementation and comparative analysis of two state-of-the-art transformer models, BERT (Bidirectional Encoder Representations from Transformers) and RoBERTa (Robustly Optimized BERT Pretraining Approach), for a multi-class emotion classification task using the provided tweet dataset. The primary objective was to fine-tune these pre-trained models and compare their performance, with a specific focus on the Macro-Averaged F1-Score, a robust metric for multi-class classification, especially with imbalanced data. Both models were fine-tuned using the Hugging Face transformers library on the raw text data. The data was found to be highly imbalanced across the six emotion classes. Against common expectations, the BERT model achieved a slightly superior Macro-Averaged F1-Score of 0.9182 on the validation set, narrowly outperforming the RoBERTa model, which scored 0.9166. This high performance across both models suggests that both architectures are highly effective at capturing the nuances of emotional language in this domain. The detailed classification reports reveal that despite the small performance difference, both models exhibit high recall even on the minority classes. The conclusion confirms BERT's marginal advantage in this specific fine-tuning setup, and future work should focus on exploring model-specific optimizations and ensemble techniques.
